# 📊 Análise de Cancelamento de Clientes (Churn)

## 📌 Sobre o projeto

Este projeto tem como objetivo analisar o comportamento de clientes de uma empresa de cartão de crédito e identificar os principais fatores que levam ao cancelamento (churn).

A análise busca gerar insights que possam apoiar estratégias de retenção e reduzir perdas financeiras.

---

## 🎯 Problema de negócio

O número de clientes cancelando seus cartões aumentou significativamente, impactando negativamente a receita da empresa.

**Pergunta principal:**

> Quais fatores influenciam o cancelamento de clientes?

---

## 📂 Dataset

O conjunto de dados contém informações sobre clientes, incluindo:

* Dados demográficos
* Comportamento de uso do cartão
* Limite de crédito
* Frequência de transações
* Status (ativo ou cancelado)

---

## 🛠️ Ferramentas utilizadas

* Python
* Pandas
* NumPy
* Matplotlib / Seaborn
* Jupyter Notebook (Google Colab)

---

## 🔎 Etapas da análise

### 1. Limpeza de dados

* Tratamento de valores nulos
* Padronização de colunas
* Conversão de tipos

### 2. Análise exploratória (EDA)

* Distribuição de clientes ativos vs cancelados
* Análise por idade, renda e limite de crédito
* Comportamento de uso

### 3. Geração de insights

* Identificação de padrões de churn
* Correlações relevantes

---

## 📈 Principais insights

* Clientes com menor uso do cartão têm maior chance de cancelamento
* Baixa frequência de transações está associada ao churn
* Clientes com limites mais baixos tendem a cancelar mais

---

## 💡 Recomendações

* Criar campanhas para aumentar o uso do cartão
* Oferecer benefícios para clientes com baixa atividade
* Revisar políticas de limite de crédito
